# Pacotes de Entrega

## Básico
- Painel diário do motor.
- Nível por setor.
- Relatório semanal curto.
- Suporte por mensagem.

## Completo
- Tudo do Básico.
- Relatório diário em texto simples.
- Ranking de ativos frágeis/estáveis.
- Revisão semanal com plano de ação.

## Sob Medida
- Tudo do Completo.
- Regras customizadas por carteira.
- Integração com fluxo interno do cliente.
- Estudo de caso focado no universo do cliente.

## Diferença entre pacotes
- `Básico`: leitura.
- `Completo`: leitura + ação operacional.
- `Sob Medida`: leitura + ação + customização de processo.

# Demo para Reunião (Guia)

## Roteiro de 15 minutos
1. Mostrar estado atual do motor em linguagem simples.
2. Mostrar setores em vermelho/amarelo/verde.
3. Mostrar “por que” de um alerta real.
4. Mostrar limites do motor sem maquiagem.
5. Fechar com proposta de piloto de 30 dias.

## Telas sugeridas
- `/app/dashboard`
- `/app/setores`

## Frases simples para usar
- “Isso não prevê o dia exato, mas mostra quando o risco estrutural está mudando.”
- “Aqui está o setor com alerta e o motivo prático para agir.”
- “Se as regras mínimas falham, a publicação é bloqueada.”

## Objetivo da reunião
- Sair com decisão de piloto e critério de sucesso combinado.
